| 字段名称 | 业务ID | 页面ID | 模块ID | 位置ID | 事件类型 |
| -- | -- | -- | -- | -- | -- |
| 描述 | 主站 main<br> 直播 live<br> 商城 mall<br> 语音包 voice<br> 运势语音 omikuji<br> | 首页 homepage<br> 列表页 list <br>详情页 detail |  | 定位 | 页面浏览 pv <br>点击 click<br> 曝光 show |
